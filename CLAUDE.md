# Baby Names Dataset

## Dataset Location
- Data files are in the `names/` directory
- Ignore `NationalReadMe.pdf` in that directory

## File Structure
- 145 files, named `yob####.txt` (e.g., `yob1880.txt`)
- CSV format with **no header row**
- Columns: `Name,Gender,Count`
- Gender values: `F` or `M`
- Line endings: Windows-style `\r\n`
